Git Version Control & GitHub

Please watch this video to better understand how to navigate Git, as well as set up your GitHub to receive initiated repositories. Below I will list some of the key commands discussed in this video.
Git Commands
  • touch ( creates a new file)
  • mkdir (makes a new directory/folder)
  • rm (removes directory or file)
  • ls ( list the files in a directory)
  • git init
  • git remote add origin [ insert github repo link]
  • git add .
  • git commit -m "your message"
  • git push -u origin main
  • git branch -b [new branch name]
  • git checkout [branch name]
  • git log ( shows the log of all saved changes in the working branch )
